# Course Project

## setup

```sh
# 安装客户端项目及依赖模块
create-react-app course
cd course && yarn eject
npm i antd moment prop-types qs react-redux react-router-dom redux redux-logger redux-promise redux-thunk axios blueimp-md5 react-swipe swipe-js-iso react-transition-group less less-loader -S

# 启动接口服务
cd server
yarn server
```
